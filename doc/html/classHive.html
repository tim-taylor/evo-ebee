Detailed Description

template<typename P>
class Hive< P >

The Hive template class This defines a concrete Hive class for a specified type of pollinator. Hence, the template class type P must be a subclass of the Pollinator class. All Hives are derived from the base class AbstractHive and inherit some functionality and interface from that.
