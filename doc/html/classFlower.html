◆ transferPollenFromPollinator()

int Flower::transferPollenFromPollinator ( PollenVector &  pollinatorStore,
int  suggestedNum 
)

Transfer some pollen grains from the specified pollinator's store to the flower's stigma, if possible. Whether of not we allow pollen from a different species to be deposited is determined by the flower's m_bPollenClogging flag. If any of the deposited pollen is from the same species, the flower will be pollinated. If we are unable to transfer as much pollen from the pollinator as requested (because of limited capacity of the stigma), then the excess is simply removed from the pollinator as if lost in the air.

Parameters
pollinatorStoreA reference to the pollinator's store from which the pollen is to be transferred
suggestedNumThe number of grains that tha pollinator would normally transfer. The actual number transferred may be less than this if the stigma has reached its maximum capacity
Returns
The number of grains transferred
