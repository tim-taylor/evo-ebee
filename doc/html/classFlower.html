int Flower::transferAntherPollenToPollinator ( PollenVector &  pollinatorStore)

Transfer some pollen grains from the anthers, if available, to the specified pollinator's store.

Parameters
pollinatorStoreA reference to the pollinator's store into which the pollen is to be transferred
Returns
The number of grains transferred

Note, no check is made as to whether the capacity of the pollinator's store has been exceeded. It is the responsibility of the pollinator to perform such a check after it has called this method.
