◆ updatePollenLandingCount()

void Pollinator::updatePollenLandingCount ( )
protected

Increment the landing count of each Pollen grain in the store. This method should be called each time a Pollinator lands on a flower. Note that this method does not remove any Pollen from the store; to remove grains that now exceed their carryover limit, call removeOldCarryoverPollen() afterwards.
