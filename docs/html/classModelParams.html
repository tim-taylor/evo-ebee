Detailed Description

The ModelParams Class holds a record of all model parameters.

All methods and variables of this class are static and can therefore be accessed by any other object in the system.

◆ getPlantTypeConfig() [2/2]

const PlantTypeConfig * ModelParams::getPlantTypeConfig ( unsigned int  speciesNum)
static

Returns a pointer to the PlantTypeConfig structure identified by the given speciesNum, which is simply the index number of the desired PlantTypeConfig in the m_PlantTypes vector. If the given speciesNum is greater than the number of plant types held in the vector, an exception is thrown. To guard against this, call getNumPlantTypes() to check how many there are before calling this method.
