Detailed Description

Note that the Colour class is used mainly for visualisation purposes. It provides methods for converting MarkerPoints to RGB values, and also for converting many named colours to RGB values. The core objects of the EvoBee model itself (pollinators, flowers, etc) use the ReflectanceInfo class to hold data about reflectance/visual properties. It is ReflectanceInfo, not Colour, that actually determines interactions between objects in the model.
